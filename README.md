# JS2 Social Media Application

## Project Description

This project is a social media application built as part of the JavaScript 2 course assignment. The application allows users to register, log in, create posts, search for posts, view profiles, and interact with a social feed using the Noroff API.

The project was built using:

- HTML
- CSS
- JavaScript (Modules)
- Noroff Social API
- GitHub Pages for deployment

---

# Features

- User registration
- User login/logout
- Authentication using access tokens
- Create new posts
- Search posts
- View profile information
- Responsive layout
- API integration
- Protected user functionality
